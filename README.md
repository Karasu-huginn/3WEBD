# Projet React Library

Application React + Vite + TypScript pour rechercher et afficher des livres.

## Architecture

```
src/
├── api/
│ ├── client.ts
│ └── index.ts
├── components/
├── hooks/
├── pages/
├── types/
├── utils/
├─ App.tsx # Routeur
└── index.css #tailwindcss
```

## Installation

```
npm install
npm run dev
```
